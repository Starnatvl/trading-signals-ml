## Research notebooks и материалы

Эта папка содержит **исследовательские (research-only)** ноутбуки и файлы.  
Они **не входят** в основной продовый пайплайн и нужны для EDA, проверки гипотез и исторического контекста.

### Структура

| Путь | Назначение |
|------|------------|
| `research/eda/Data_Exploration_dataset_rework.ipynb` | Большой EDA по `dataset_rework`: распределения, пропуски, интервалы, базовые проверки качества данных. |
| `research/eda/eda_summary_dataset_rework.txt` | Краткое текстовое резюме выводов из EDA‑ноутбука. |
| `research/hypotheses/Feature_Engineering_and_Hypotheses.ipynb` | Черновые гипотезы по фичам и таргетам, графики и эксперименты. Исторический контекст для решений в `docs/` и `01_data_prep/01_Load_And_Prepare_Data.ipynb`. |

### Как этим пользоваться

- Для **воспроизводимого пайплайна** достаточно запускать ноутбуки, начиная с `01_data_prep/01_Load_And_Prepare_Data.ipynb`, затем `02_targets/03_...`, `03_features/...`, `04_models/...`, `05_experiments/...`.  
- `research/*` можно открывать, когда нужно вспомнить, **почему** были приняты те или иные решения (фичи, target, split и т.п.), либо для генерации новых гипотез.
